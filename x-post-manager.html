⚠️ このファイルはローカルで開かれています。
ブックマークレットは https:// ページから file:// へのリダイレクトをブロックするため、このままでは動作しません。
下の手順に従って GitHub Pages にアップロードしてください。

🔧 セットアップ（初回のみ）

1
このファイルを GitHub Pages に置く
すでに hajime085.github.io をお持ちなので、リポジトリに
このファイルを x-post-manager.html という名前でアップロードするだけです。

アップロード後にアクセスするURL:
https://hajime085.github.io/affiliate-tool/x-post-manager.html
（リポジトリ名に合わせて変更してください）
2
GitHub Pages のURLでページを開き、ブックマークレットをドラッグ
GitHub Pages の URL（https://）でこのページを開くと、
下のブックマークレットボタンが有効になります。
ボタンをブックマークバーへドラッグしてください。
X Post を保存
← このボタンをブックマークバーへドラッグ
※ https:// のURLで開いているときのみ有効
3
X でポストを保存する
X の個別ポストページ（x.com/～/status/～）で
ブックマークバーの「X Post を保存」を押すと、このページに自動で追加されます。
